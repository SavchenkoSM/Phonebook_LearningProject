# Phonebook [LearningProject]
### :telephone_receiver: :blue_book: Phonebook App 
* Each contact record consists **4** fields: 
  * Last name;
  * First Name;
  * Middle Name;
  * Phone number. 
* **Functionality**:
  * search a record by its fields;
  * add a record;
  * edit a record;
  * delete a record. 
* **Data storage**: all records are stored in the local SQLite database.
